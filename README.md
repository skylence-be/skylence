# skylence

Public issue tracker for **skylence**: the agent runtime that glues the sky stack together. One in-process loop that turns plans into verified changes, with skyline as hands, skybox as memory, and skycastle as keys.

In development, not yet publicly released. Development happens in a private repository; this tracker is open early so feedback and requests have a home.

Part of the [sky stack](https://github.com/skylence-be), alongside [skyline](https://github.com/skylence-be/skyline).
