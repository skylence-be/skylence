# Skylence

Claude Code harness builder. Go binary that wires GitHub webhooks, workflows, and the `claude` CLI into a daemon you run locally.

This repository hosts the public release artifacts. The source is private.

## Links

- Website — https://skylence.be
- Documentation — https://docs.skylence.be

## Installation

Release binaries are published here under [Releases](../../releases). `sky update` on an existing install pulls from the same feed.
